ratbag-emu - HID device emulator
================================

.. image:: _static/ratbag.svg
   :width: 30%
   :align: right

ratbag-emu is a HID device emulator built on top if the UHID API of the Linux
kernel. It was developed with the intended use of emulating real devices in a
CI scenario.
